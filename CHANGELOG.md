# 📋 Changelog

## [Unreleased] - 2025-05-16

### ✨ Adicionado
- Tela de carregamento centralizada (`LoadingScreen.tsx`) com animação de fade-in da logo.
- Barra de progresso animada durante o carregamento da página.
- Ícones de busca, usuário e carrinho (FaSearch, FaUser, FaShoppingCart) adicionados lado a lado na navbar.
- Ícone vetorial desenhado representando a marca MiniStore (celular, relógio eletrônico, etc).
- Substituição do FaMobileAlt pelo novo logo vetorial personalizado.

### 🛠️ Modificado
- `Nav` agora é responsivo com media query a partir de 820px.
- Submenu fecha automaticamente ao clicar fora do menu.
- Efeitos visuais adicionados ao botão de menu mobile (`MobileMenuButton`) com transições animadas.

### 📁 Organização
- Código mais modular com separação da tela de carregamento e importação da logo SVG.
- Comentários antigos removidos ou ajustados para clareza.
